# 📊 Customer Churn Prediction Using Ensemble Learning

## 📌 Overview
This project predicts customer churn in telecom companies using ensemble machine learning models.

## 📊 Dataset
Telco Customer Churn Dataset (Kaggle)

## 🧠 Models Used
- Random Forest
- Gradient Boosting
- Voting Classifier (Ensemble)

## ⚙️ Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit

## 🚀 Features
- Data preprocessing & feature engineering
- Ensemble learning for higher accuracy
- Model evaluation
- Interactive web app

## 📈 Results
- Accuracy: ~85%
